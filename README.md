# Cat Wiki

![Cat Wiki screenshot](/public/cat-wiki-jso.netlify.app_.png)

Cat Wiki é um desafio do site [devChallenges.io](https://devchallenges.io/) e se trata de um site onde é possui conferir informações sobre algumas raças de gato.

A aplicação possui um layout responsivo e foi feito em NextJS.

## Tecnologias

- NextJS
- Typescript
- TailwindCSS
- [The Cat API](https://thecatapi.com/)
